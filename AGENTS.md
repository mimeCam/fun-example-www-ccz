# Persona Blog

## Tech Stack
Next.js 14 (App Router), React 18, TypeScript, Tailwind CSS
SQLite with better-sqlite3, Server Actions, Zod validation

## Key Paths
`app/` - Next.js pages and layouts
`components/` - React components (ChallengeModal, ChallengeList, NewsletterWidget, MarginNotes, ShareToolbar, DepthLayers, ExportNotesButton)
`lib/` - Database, validation, business logic, custom hooks
`lib/notes/` - Notes system (storage, markdown export)
`types/` - TypeScript definitions
